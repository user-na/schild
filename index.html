Schild (Einheit)

Definition

Die Schild ist eine dimensionslose, normierte Einheit zur Beschreibung der Belastung, die eine IT-Komponente (z. B. CPU, Netzwerkinterface, Storage-Device) während eines definierten Betriebsintervalls erfährt. Ein Schild-Wert kombiniert mehrere Metriken wie CPU-Auslastung, Temperatur-Index, Latenz-Score und I/O-Last in einem vergleichbaren Maßstab.

Historie

Als fiktive Einheit wurde die Schild in dieser Demo entwickelt, um eine einfache und vergleichbare Kennzahl für Belastungstests zu demonstrieren. In realen Umgebungen existieren ähnliche zusammengesetzte Indizes, die aber projekt- oder herstellerabhängig variieren.

Definition der Einheit

Formal lässt sich der Schild als gewichteter Index darstellen:

S = w1 * C + w2 * T + w3 * L + w4 * I

wobei:

  • C = CPU-Auslastungsanteil (0–1)
  • T = Temperatur-Normalisierter Index (0–1)
  • L = Latenz-Score (0–1)
  • I = I/O-Last-Norm (0–1)

Die Gewichtungen w1..w4 werden projektbezogen festgelegt und normieren den Index typischerweise auf einen Maximalwert von 100 S (oder 1.0 in normierter Form). Für diese Beispielseite nehmen wir eine Normierung, bei der 1 Schild = 1 Einheit des kombinierten Index (interpretierbar als 1 S = 1·Index-Punkt).

Anwendungen

Die Schild-Einheit kann verwendet werden für:

  • Lasttests und Benchmarking, um Komponenten unter vergleichbaren Bedingungen zu bewerten.
  • SLAs: Ableiten von Schwellenwerten für Grenzwerte (z. B. kritisch ab 80 S).
  • Monitoring-Dashboards: Aggregation heterogener Metriken in einem einzigen, leicht verständlichen Wert.

Beispiele

Beispielrechnung (gewichtete Normierung):

Gegeben: C=0.6, T=0.4, L=0.2, I=0.5 mit Gewichten w1=0.4, w2=0.2, w3=0.2, w4=0.2

S = 0.4*0.6 + 0.2*0.4 + 0.2*0.2 + 0.2*0.5 = 0.24 + 0.08 + 0.04 + 0.10 = 0.46 → 46 S (bei Skalierung auf 0–100)
